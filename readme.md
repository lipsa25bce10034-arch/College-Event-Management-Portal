# College Event Management System

## Project Overview
The *College Event Management System* is a web-based application designed to streamline the organization, management, and participation of college events. This system allows administrators to manage events, students to register and participate, and faculty to monitor event progress efficiently.

---

## Features

### Admin Features
- Add, edit, and delete events.
- Manage student and faculty registrations.
- View reports of event participation.
- Approve or reject event registrations.

### Student Features
- Register and login to the system.
- View available events.
- Register for events and receive notifications.
- Check participation history.

### Faculty Features
- Monitor student registrations.
- Approve participation requests.
- Generate event reports and feedback.

---

## Technologies Used
- *Frontend:* HTML
- *Backend:* Python 
- *Database:* MySQL
- *Tools:* VS Code, GitHub
